Industrial Controller Settings

The addon allows you to select any channel in the target for use as a feedback signal, output signal, and setpoint signal. It also allows you to set limits on what is a valid setpoint and output range. Once this has been configured, go to the Industrial Parameters section.

What do you want to do?

Enter the setpoint signal settings

By default, a setpoint channel will be automatically created for each controller. This channel is in the Channels folder. You can use this channel to update the setpoint to the controller. However, you can also specify that the setpoint signal come from another location (ie. a model). To do this, Select the Use External Setpoint field. This will show a channel browser. Select the browse button next to the Setpoint field and select the desired signal. You can then enter in a valid setpoint range with the Setpoint Max (EU) and Setpoint Min (EU) fields.

Select a feedback signal

Select the browse button next to the Feedback field and select the desired signal.

Enter the output signal settings

The industrial controller is basically an On/Off controller. It will look at the setpoint and the feedback signal and set the output either on (activated) or off (deactivated). You can set the actual output level for each state in Activated Level (EU) and Deactivated Level (EU) respectively. Then, select the browse button next to the Output field and select the desired signal.

Delete item

To delete, right-click the object in the configuration tree and select Delete from the shortcut menu.
