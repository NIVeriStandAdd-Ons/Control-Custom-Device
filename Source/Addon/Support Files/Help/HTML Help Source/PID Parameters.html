PID Parameters

This addon supports configuring default PID gains and gain schedule if needed.

What do you want to do?

Enable gain scheduling

If you need gain scheduling, select the Enable Gain Scheduling field. This will show the channel browser. Then, select the gain schedule signal by using the browse button next to the Gain Schedule Signal field.

Enter default PID gains

To enter the default PID gains, single click under P, I, and D to type in a value. When the settings are correct, select Apply. If you would like to revert back to the last applied PID gains, select Reset. The Gain Schedule Value is ingored when gain scheduling is disabled. If gain scheduling is enabled, you can add additional rows by right clicking on the PID Settings table and select Add Row. Additionally, you can remove a row by selecting Delete Row.

Note: If you are using gain scheduling, make sure the lowest gain schedule value is first in the list and the greatest gain schedule value is last in the list.

Enable live PID gain updates

If you want to be able to change the PID parameters on the fly using the PID tool, select Enable Live Updates.

Note: Any gain changes that are applied on the fly will be lost on the next deployment. If you want to keep your changes, you must modify the default settings. You can do this by exporting desired changes in the PID tool and importing them to the PID Parameters section using the Import button.
