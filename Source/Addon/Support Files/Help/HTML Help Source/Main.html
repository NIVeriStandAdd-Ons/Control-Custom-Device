Simple PID Addon

This addon allows you to do two types of control methods. The first is a single mode PID control with gain scheduling and bumbless manual control. The second is simple on and off control for industrial actuators.

What do you want to do?

Add a PID controller

To use this addon, first add as many PID controllers as are needed. Do this by selecting the Add PID button or by right clicking on the addon and selecting Add PID. Then configure each PID Controller.

Add an industrial controller

To use this addon, first add as many industrial controllers as are needed. Do this by selecting the Add Industrial button or by right clicking on the addon and selecting Add Industrial. Then configure each industrial controller.

Delete item

To delete, right-click the object in the configuration tree and select Delete from the shortcut menu.

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.
